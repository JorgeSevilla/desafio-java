If - else if - else

Adivina el número: Genera un número aleatorio entre 1 y 100. Pide al usuario que adivine el número en el menor número de intentos posible. Utiliza if else elseif para dar pistas como "mayor", "menor", "adivinaste".
Clasifica triángulos: Pide al usuario que ingrese la longitud de los tres lados de un triángulo. Usa if else elseif para determinar si es equilátero, isósceles, escaleno o no es un triángulo.
Calculadora de descuentos: Crea una calculadora de descuentos basada en el total de la compra y el código de descuento ingresado. Aplica diferentes descuentos (if), descuentos especiales para clientes VIP (elseif), o precio sin descuento (else).
Juego de piedra, papel o tijera: Crea un programa que juegue piedra, papel o tijera contra el usuario. Utiliza if else elseif para determinar el ganador y mostrar mensajes como "ganaste", "perdiste", o "empate".
Retos de simulación y texto:

Asistente de vestimenta: Pide al usuario información sobre el clima y la ocasión. Utiliza if else elseif para sugerirle diferentes prendas de vestir en función de las condiciones.
Adivina la fruta: Describe una fruta con pistas sensoriales (color, sabor, textura). Deja que el usuario adivine la fruta usando preguntas con respuestas sí o no. Utiliza if else elseif para responder a las preguntas y revelar la fruta cuando se adivine.
Analizador de emociones: Lee una frase del usuario e identificala como positiva, negativa o neutral. Utiliza if else elseif para analizar palabras clave relacionadas a las emociones.
Generador de historias: Crea una historia interactiva donde el usuario toma decisiones en puntos clave. Utiliza if else elseif para ramificar la historia en diferentes caminos basedo en las elecciones del usuario.
Retos creativos:

Traductor de emojis: Pide al usuario una frase y traduce cada palabra a un emoji similar. Crea reglas divertidas para la traducción utilizando if else elseif (por ejemplo, "perro" a , "flor" a ).
Creador de rimas: Introduce una palabra y genera rimas usando un diccionario de rimas y if else elseif para encontrar palabras con sonidos similares.
